## 1번 for문 사용해서 구구단 짜기

### 예상되는 결과값
```
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
...(생략)
3 x 1 = 3
3 x 2 = 6
...(생략)
9 x 8 = 72
9 x 9 = 81
```

### 정답
```js
for (let i = 2; i <= 9; i++) {
    for (let j = 1; j <= 9; j++) {
        console.log(`${i} X ${j} = ${i * j}`);
    }
}
```

## 2번 for문 사용해서 탑 찍기

### 예상되는 결과값
```
*
**
***
****
*****
******
*******
```

### 2번 정답
```js
for (let i = 0; i < 7; i++) {
    let str = "";
    for (let j = 0; j < i + 1; j++) {
        str += "*";
    }
    console.log(str);
}
```

## 3번 for문 사용해서 탑 찍기2 (보너스)

### 예상되는 결과값
```
     *
    ***
   *****
  *******
 *********
***********
```

### 3번 정답
```js
for (let i = 0; i < 6; i++) {  
    let str = "";
    for (let j = 5 - i; j >= 0; j--)
        str += " ";
    for (let j = 0; j < i * 2 + 1; j++)
        str += "*";
    console.log(str);
}
```
